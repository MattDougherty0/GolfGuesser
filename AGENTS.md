# Agent Guidelines

## Git Branch Workflow

**Before starting any coding task:**

1. **Check current branch** – Confirm we are not on `main`.
2. **Create a new branch** – If on `main`, create and switch to a new branch first (e.g. `git checkout -b feature/description` or `git checkout -b fix/issue`).

Never make edits directly in `main`. All work should happen in a feature or fix branch.
